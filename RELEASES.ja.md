## v0.3.0 — システム通知 & ミュート検出

### ✨ 新機能

| 機能 | 説明 | 提供 |
|------|------|------|
| システム通知ポップアップ | タスク完了時にサウンドと同時に視覚的通知を表示 | — |
| スマート自動モード | `notify: "auto"` がシステムのミュートを検出し、無音時のみ通知 | — |
| 通知の設定 | `notify: "on"`（常時）、`"off"`（無効）、`"auto"`（スマート） | — |

### 🛠️ 変更点

| 変更 | 説明 | 提供 |
|------|------|------|
| `loudness` 依存を追加 | 信頼性の高いクロスプラットフォームミュート検出、ネイティブビルド不要 | — |
| COM/PowerShell ハックを削除 | 脆弱な Windows オーディオ検出を `loudness` に置き換え | — |

### 📦 インストール

```bash
npm install -g vibe-notify       # macOS / Linux
npm install -g vibe-notify       # Windows（追加手順不要）
npx vibe-notify                  # インストール不要で実行
```
